Python is one of my strongest languages. I've worked on many projects using this language and have even used it to help teach programming concepts.

One project I've worked on using Python was creating a DNS server. This project was to use socket-based programing to mimic a real world DNS server system. It was built to allow a controlable amount of clients to connect to the server, as well as to be expandable if more servers were needed. The version that this project used had 5 total servers to mimic a local DNS server, a Root DNS Server, and then a Server for .com, .gov, and .org urls.

I've also used Python for API work, specically with the Google Sheets API.
